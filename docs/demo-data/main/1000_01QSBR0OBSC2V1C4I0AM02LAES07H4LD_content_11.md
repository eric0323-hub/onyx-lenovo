### Symptom

Numlock is enabled, but numeric keypad does not work.

### Applicable Brands

ideapad

### Operating Systems

* Windows 10
* Windows 11

### Solution

1. Usually, there is a NumLock indicator light on the keyboard. Check whether the indicator light is on, and inspect the keyboard for any stuck keys.
2. Navigate to **Control Panel** -> **Ease of Access Center** (or search for and select **Control Panel**, then select **Ease of Access Center**). For more information about Control Panel, see [Where is Control Panel](https://support.microsoft.com/en-us/help/13764/windows-where-is-control-panel).  
   ![Control Panel](https://download.lenovo.com/km/media/images/HT503086/control_20220328170703545.png)
3. Scroll down, then click **Make the mouse easier to use** or **Change how your mouse works**.  
   ![Mouse](https://download.lenovo.com/km/media/images/HT503086/easeofaccess1_20161216095012.PNG)  
   
   Windows 11  
   ![Change how your mouse works](https://download.lenovo.com/km/media/images/HT503086/Windows11changehowmouse_20260127134849390.png)
4. Find the **Control the mouse with the keyboard** section. Clear the box next to **Turn ON mouse keys**. Click **OK**.  
   ![Control the mouse](https://download.lenovo.com/km/media/images/HT503086/mouse easier to use_20230822171815397.png)
