### Symptom

When trying to restore the system to factory settings using System reset option, it prompts the following error message "There was a problem resetting your PC".

![ideapad 100 recover failure](https://download.lenovo.com/km/media/images/HT104563/64f38764b5ba4d01bfb5a447cd93dd33.jpg)

### Affected brands

Lenovo laptops

### Affected systems

Ideapad100s-14IBR shipped before November 2015

### Applied OS

Windows 10

### Solution

1. Download [winpe64\_repairbcd.zip](https://download.lenovo.com/consumer/hnt/winpe64_repairbcd.zip). After downloading completes, verify MD5. The correct MD5 should be F48E462B673A1EAF194A00DBD82ECFA9.  
2. Make a bootable USB disk following [PBR function repair SOP](https://download.lenovo.com/consumer/hnt/sop_pbr_function_recover.doc) .  
3. Following [PBR function repair SOP](https://download.lenovo.com/consumer/hnt/sop_pbr_function_recover.doc)  to resolve the above recovery failure.

**Note:**   
Please never plug in SD/TF card during recovery or PBR repair process.
