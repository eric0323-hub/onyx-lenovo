## Symptom

Lenovo IdeaPad 5 2-in-1 14AHP9 Keyboard is malfunctioning: Some keys are not working.

## Applicable Systems

Lenovo IdeaPad 5 2-in-1 14AHP9

## Solution

The FSE’s connecting keyboard cable connector is upside down), causing key malfunctions.

![](https://ukm.lenovo.com/prod-api/file/download?key=accessory/6fe825fb8a1f4a159941d022782b5988-1764665134340.jpg&name=Screenshot 2025-12-02 163012..jpg)

[Keyboard malfunction - when connecting cable upside down.mp4](https://drive.google.com/file/d/1YBIuKdXPaDHBOduGn69dvSJqSgLod5uE/view?usp=sharing)

Re-route the cable correctly as shown in the image below.
