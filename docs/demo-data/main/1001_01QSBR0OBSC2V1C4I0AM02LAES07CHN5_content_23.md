To troubleshoot USB keyboard problems, use the following steps:

For information about the fingerprint reader included on some keyboards, please refer to [Fingerprint Reader tips](https://support.lenovo.com/solutions/HT037692).

The following procedures are primarily used when the keyboard is not working, a keyboard error during POST is developed, or if the computer will not wake after entering suspend/sleep mode.

**Note:** Some of the steps may not apply to your computer.

1. Check that the keyboard and mouse are correctly connected and seated all the way in the port.
2. Disconnect all extension cables and switchboxes connected between the keyboard and the computer. If the computer is connected to a monitor, mouse, and keyboard through a Keyboard-Video-Mouse (KVM) switch, or console selector switch, disconnect the switch and connect the monitor, mouse, and keyboard directly to the computer. If this solves the issue, contact the switch manufacturer for support and compatibility on the switch. Some switches can have a firmware update applied for the most common fixes.
3. Check the keyboard connector for bent, or other damage to the connector.
4. Verify that the keyboard works in the BIOS utility. If so, load the default settings. Then save the settings, and exit the utility. For more information, see [Recommended ways to enter BIOS - ThinkPad, ThinkCentre, ThinkStation](http://support.lenovo.com/solutions/ht500222).
5. If the keyboard works during the POST or in the BIOS utility, but not in the operating system, check software logs for device driver errors. Contact your software vendor for more support in configuring the I/O devices.
6. Verify that the keyboard port is working by testing another keyboard on the computer. If the replacement keyboard works, the other keyboard is defective.
7. Verify that the keyboard is working by testing the keyboard on another computer. If it does not work on a second computer, the keyboard is defective.
9. If the keyboard is an option, verify that the computer meets the minimum operating system and hardware requirements of the keyboard.
10. For Rapid Access Keyboards (RAK), ensure that the most current drivers are installed and functioning properly.
11. If these steps have not solved your problem,[Contact Support](https://pcsupport.lenovo.com/).

**Note:** Signal cable extensions and KVM switch boxes are not supported. These devices can cause poor video quality, excessive voltage drops, and permanent system damage.
