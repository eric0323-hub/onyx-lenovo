### Applicable Brands

Lenovo Tablet

### Applicable Systems

YOGA Tab 3 Plus

### Operating Systems

Android 6.0 Marshmallow

### Solution

Use the following steps:

1. Touch **Advanced Menu** on Home screen.  
      
    ![Lenovo YT-X703F - Use Dolby - Advanced Menu](https://download.lenovo.com/km/migrationImages/HT502842/Lenovo_yt_x703f_059_001_20161110060105.jpg)  
    Fig.1
2. Tap **Dolby**.  
      
    ![Lenovo YT-X703F - Use Dolby - Dolby](https://download.lenovo.com/km/migrationImages/HT502881/Lenovo_yt_x703f_029_002_20161115082013.JPG)  
    Fig.2
3. Tap **Power icon** at top left.  
      
    ![Lenovo YT-X703F - Use Dolby - Power icon](https://download.lenovo.com/km/migrationImages/HT502881/Lenovo_yt_x703f_029_003_20161115082104.JPG)  
    Fig.3
4. **DOLBY ATMOS** provides four original modes (**Movie**/**Music**/**Game**/**Voice**). Select a preferred mode.  
      
    ![Lenovo YT-X703F - Use Dolby - Movie/Music/Game/Voice](https://download.lenovo.com/km/migrationImages/HT502881/Lenovo_yt_x703f_029_004_20161115082328.JPG)  
    Fig.4
5. Or scroll downwards and select **Custom** mode.  
      
    ![Lenovo YT-X703F - Use Dolby - Custom](https://download.lenovo.com/km/migrationImages/HT502881/Lenovo_yt_x703f_029_005_20161115082406.JPG)  
    Fig.5
6. Adjust **Volume Leveler**, **Dialogue Enhancer**, and **Surround Virtualizer** as preferred.  
      
    ![Lenovo YT-X703F - Use Dolby - Volume Leveler, Dialogue Enhancer, and Surround Virtualizer](https://download.lenovo.com/km/migrationImages/HT502881/Lenovo_yt_x703f_029_006_20161115082502.JPG)  
    Fig.6
