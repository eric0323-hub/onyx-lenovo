## Symptom

* The widgets are unavailable in PC mode.
* Even when long-pressing the Home screen, only the icon reordering command is visible and there is no option to add widgets.
  
  ![](https://ukm.lenovo.com/prod-api/file/download?key=ukm2/accessory/14c7c9175fe141bda771e55974fc3cb9-1775641399133.png&name=image..png)

## Applicable systems

Lenovo Yoga Tab Plus - TB520FU (Type: ZAEG)

## Solution

* The widgets can’t be added in PC mode.
* That is work as design.
