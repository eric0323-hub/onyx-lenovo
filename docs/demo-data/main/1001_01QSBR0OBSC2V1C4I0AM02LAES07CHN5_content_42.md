### Symptom

When touching the screen, the system will enter sleep mode.

### Applicable Brands

ideapad

### Applicable Systems

* Yoga 710-14ISK
* Yoga 710-14IKB

### Operating Systems

Windows 10 (64-bit)

### Solution

Download the Video and Chipset drivers for the unit from the support page. Make sure to reboot after an update on the unit.

If this has no effect, try a static discharge. Use the following steps to finish the static discharge.

If the battery is internal, skip the remove/add battery steps.

1. Turn the unit off.
2. Remove the battery and the main adapter.
3. Press and hold the front power button for 30 seconds.
4. Add the battery and the main adapter back.
5. Turn the unit on as normal.
