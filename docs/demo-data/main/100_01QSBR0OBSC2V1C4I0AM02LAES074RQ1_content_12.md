### Symptom

The ThinkPad Yoga 460/P40 system experiences LCD flickering when resuming from the Power State S3 (Sleep mode) or when the lid is closed.

### Applicable Brands

ThinkPad

### Applicable Systems

* ThinkPad Yoga 460
* ThinkPad Yoga P40

### System Is Configured With

WQHD panel (LG vendor)

### Operating Systems

Microsoft Windows 10

### Limitations

There is a PSR timing issue on LG vendor panels.  Intel cannot modify this PSR timing to max value per LG’s requirement.

### Workaround

To disable the Display Power Saving Technology when your system experiences this failure, choose the menu in the Intel HD Graphics Control Panel and select **Disable**.
