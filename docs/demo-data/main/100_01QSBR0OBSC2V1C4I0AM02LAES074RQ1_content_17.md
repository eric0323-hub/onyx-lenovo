# 🛠️ Troubleshooting Power Issues on ThinkPad T14 Gen 5 with Dock

![](https://ukm.lenovo.com/prod-api/file/download?key=2025/04/29/Lenovo_Logo_20250429174901A001.png)
## 📌 Issue Overview

Users may encounter a problem where the ThinkPad T14 Gen 5 (AMD) cannot be powered on using the dock's power button when the lid is closed, particularly when running BIOS version 1.11.

## ⚙️ Solution Steps

To resolve this issue, follow the steps below:

1. **Roll Back BIOS Version**
   
   * Downgrade the BIOS from version 1.11 to version 1.10.
2. **Access Rollback Instructions**
   
   * For detailed instructions on how to roll back the BIOS update, visit the following link:  
     [How To Roll Back BIOS Update](https://ukm.lenovo.com/#/homePage/homeDetail?uid=0x44efcb8b&title=How+To+Roll+Back+BIOS+Update&back=/homePage/searchDetail)
3. **Download BIOS Version 1.10**
   
   * Download the BIOS version 1.10 from the Lenovo support site:  
     [Download BIOS R2LUJ29W.exe](https://download.lenovo.com/pccbbs/mobiles/R2LUJ29W.exe)
4. **Install the BIOS**
   
   * Follow the instructions provided in the rollback guide to install the downloaded BIOS version.

## 🎯 Expected Outcome

After completing these steps, the power button on the dock should function correctly, allowing you to power on the ThinkPad T14 Gen 5 with the lid closed.

For further assistance, please consult the Lenovo support documentation or contact technical support.
