### Symptom

Windows has a black screen when coming out of suspend or hibernate. For power related issues, see [Troubleshooting No Power Issues](https://support.lenovo.com/solutions/ht510410).

### Operating Systems

* Windows 10
* Windows 11

### Solution

Press the Windows key + Shift + Ctrl + B at the same time. This should bring the screen back. Then apply any Windows updates.

If this does not work, see the following Microsoft link for more information: [Toubleshoot black screen or blank screen errors](https://support.microsoft.com/windows/troubleshooting-blank-screens-in-windows-51ef7b96-47cb-b454-fcab-fac643784457)

Starting the PC in safe mode can help narrow down the issue. Safe mode loads default settings and basic device drivers. See [Start your PC in safe mode in Windows 10 or Windows 11](https://support.microsoft.com/windows/start-your-pc-in-safe-mode-in-windows-10-92c27cff-db89-8644-1ce4-b3e5e56fe234?ui=en-us&rs=en-us&ad=us) for more information.

A black screen after suspending or entering hibernation can also be caused by other factors:

* Check drivers for the keyboard and mouse. Keep these drivers up to date.
* Examine your power settings. Some options, such as Fast Startup, can result in a black screen when the system is waking up.
* Watch out for third-party apps. Unknown programs may cause issues.

**Note:** If the PC is using an external monitor, make sure the monitor cable is not loose.

Try the following steps to resolve this issue:

#### Step 1: Check Whether your Device is Allowed to Wake the Computer

1. Right-click **Start** (or search for and select **Device Manager**).
2. Select **Device Manager**.  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT508747/devicemanstart_20241106134331467.png)  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT508747/Windows11Devicemansearch_20260109142911149.png)
3. Right-click **HID Keyboard/mouse**. Select **Properties**.  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT508747/324131_20230810015332456.png)
4. Select **Power Management** tab, then check **Allow this device to wake the computer**.


#### Step 2: Apply Windows Updates

Use the following steps to apply Windows Updates. For more information, see [How to run Windows Updates - Windows 10 and Windows 11](https://support.lenovo.com/solutions/HT515362).

1. Select **Start** (for Windows 11, use **Start**, **Settings**, **Windows Updates**).
2. Select **Settings**.  
   ![Settings](https://download.lenovo.com/km/media/images/HT508747/Settings4_20240313183935993.png)
3. Select **Update & Security**.  
   ![Update and Security](https://download.lenovo.com/km/media/images/HT508747/updateandsecurity_2024081516044071.png)
4. Select **Windows Updates** and **Check for updates**.  
   ![Windows Update](https://download.lenovo.com/km/media/images/HT510664/Windowsupdates1_20220628153350639.png)


#### Step 3: Update drivers using Automatic Update

Update drivers from the Product Home page.

1. Go to the [support site](https://pcsupport.lenovo.com/) and provide product information or use **Detect Product**.
2. Select **Drivers & Software**.  
   ![Drivers and Software](https://download.lenovo.com/km/media/images/HT508747/driversandsoftware2024_20240626120716225.png)
3. Use the **Scan Now** option under **Automatic Update** to update drivers.  
   ![Automatic Update](https://download.lenovo.com/km/media/images/HT508747/automaticdriverupdateJan2023_20230207162025321.png)


#### Step 4: Update Mouse or Keyboard drivers from Device Manager

1. Search for and open **Device Manager**.  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT508747/devicemansearch_20230323153208624.png)
2. Find the keyboard or mouse driver. Right-click the driver and **Update driver**.  
   ![Update drivers](https://download.lenovo.com/km/media/images/HT508747/keyboardsdm_2023032315340166.png)
