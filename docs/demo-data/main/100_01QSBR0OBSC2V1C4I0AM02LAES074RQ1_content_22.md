### Symptom

The display is abnormal when waking up from sleep and hibernate on a daisy chained monitor.

This may include the following behavior.

* The display shows a black screen and the system hangs when using four daisy chained monitors after waking from sleep.
* The display shows a black screen on one monitor after waking up from hibernate.

### Applicable Brands

ThinkStation

### Applicable Systems

ThinkStation P340

### System Is Configured With

* NVIDIA Quadro P400
* NVIDIA Quadro P620
* NVIDIA Quadro P1000
* NVIDIA Quadro P2200
* NVIDIA Quadro RTX 4000
* NVIDIA Quadro RTX 5000

### Operating Systems

Windows 10 64 bit

### Solution

Do NOT replace any hardware. A driver will be released to resolve this issue in April 2021.
