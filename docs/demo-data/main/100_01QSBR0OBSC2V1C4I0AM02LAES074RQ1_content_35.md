## Symptom

Device compatible with Secure Launch will not shutdown or hibernate after Windows KB KB5073455.

## Applicable Systems

* ThinkPad
  
  ThinkCentre
  
  ThinkStation

## Operating Systems

Windows 11 with KB KB5073455

## Solution

Determine if device is using Secure Launch.

* Open System Information
* Look for the **Secure Launch** tag in the **Virtualization-Based Security** section.

## Workaround

Uninstall KB5073455.
