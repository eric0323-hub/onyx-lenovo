### Description

Currently, some of customer from forum or any other channel report that their Lenovo Tab4 10 Plus (Lenovo TB-X704 seem to be having an issue with 5G WIFI.The tablet connects to WIFI access point, and internet will work for a few minutes, but shortly thereafter the internet will stop working (although the tablet will stay connected to the WIFI network). After a while I'll get the "X" symbol next to the WIFI icon.

![](https://download.lenovo.com/km/media/images/HT508225/1_20190403012015680.jpg)

Disconnecting and reconnecting will get internet working again, but after a few minutes the same thing happens.

This issue may only happen on 5GHz WIFI wireless router and 2.4GHz condition no same failure.

### Applicable Systems

TB-X704L;TB-X704F

### Solution

This issue was caused by Qualcomm WIFI vulnerability patch
