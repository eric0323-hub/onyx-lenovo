# 门店设备更换路由器后持续离线

## 背景

某连锁门店升级宽带并更换了无线路由器。收银旁的智能终端可以开机，但后台设备列表显示离线，门店员工无法同步库存和订单状态。

## 常见原因

新路由器的 WiFi 名称与旧配置不一致，设备仍在尝试连接原网络。部分路由器默认开启 5GHz 优先，老型号终端只支持 2.4GHz，因此会出现连接失败。也有门店开启了访客网络隔离，导致终端无法访问局域网网关。

## 排查步骤

首先在设备设置页查看当前连接的 SSID、IP 地址、网关和 DNS。若 IP 为 `169.254` 开头，说明设备没有从 DHCP 获取地址。应重新输入 WiFi 密码，并确保设备连接的是 2.4GHz 网络。

如果设备已获得内网地址但后台仍显示离线，需要检查路由器是否屏蔽了外网访问。可在同一网络下用手机访问业务域名，确认 HTTPS 连接是否正常。若门店使用企业防火墙，还要放通设备访问云端服务所需的端口。

## 交付标准

设备重新上线后，后台状态应在 2 分钟内变为在线。工单中需要记录新 SSID、设备 IP、网关、DNS 和是否调整过路由器隔离策略。
