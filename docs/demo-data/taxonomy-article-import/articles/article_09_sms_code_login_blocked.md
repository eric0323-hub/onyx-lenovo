# 用户收不到短信验证码无法登录

## 背景

客户员工更换手机后无法登录企业账号。输入手机号后页面提示验证码已发送，但用户连续三次没有收到短信。其他同事同一时间可以正常登录。

## 排查方向

客服应先确认手机号是否与企业账号绑定一致，避免用户输入旧号码。然后检查该号码是否触发发送频率限制、是否被运营商拦截，或是否存在国际区号选择错误。

## 处理步骤

如果账号信息正确，可让用户等待 5 分钟后重新获取验证码，并提醒检查短信拦截箱。若仍然无法收到，客服可以通过管理员确认用户身份后，临时改用邮箱验证码或一次性登录链接。

对于多次错误输入密码后被登录保护拦截的账号，需要先解除风控限制，再重新发送验证码。解除前必须核对企业管理员授权，避免把登录能力开放给非本人。

## 工单记录

记录用户手机号后四位、企业名称、验证码发送时间、失败次数和是否启用替代验证方式。若同一运营商号码集中失败，应升级给短信服务供应商排查通道状态。
