# 后台首页核心功能新手问答

## 背景

很多新用户首次进入后台时，不确定首页卡片、筛选器和快捷入口分别用于什么场景。本文整理一线客服最常收到的基础操作问题。

## 首页数据从哪里来

首页展示的是当前账号有权限查看的数据。如果用户只属于某个门店，首页不会显示其他门店的订单和库存。数据每隔数分钟刷新一次，手动点击刷新按钮可以立即拉取最新统计。

## 如何进入常用功能

左侧导航包含订单、客户、库存和设置。新用户建议先完成个人信息、通知偏好和默认门店设置。常用页面可以加入收藏，之后会出现在首页快捷入口。

## 为什么看不到某些按钮

按钮是否显示取决于账号权限和当前套餐。如果用户无法导出报表，可能是没有导出权限，也可能是套餐未开通高级报表功能。遇到这种情况，应联系企业管理员确认配置。

## 入门建议

新用户可以先使用示例数据熟悉操作，再切换到真实业务数据。完成前三天试用后，建议参加一次线上培训，减少后续误操作。
