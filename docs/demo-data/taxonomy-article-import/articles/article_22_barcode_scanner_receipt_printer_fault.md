# 笔记本键盘背光分区异常处理记录

## 问题背景

部分高端笔记本在更新固件后出现键盘背光异常：键盘本身可以正常输入，单个按键没有失灵、连键或卡键，USB 接口和外接设备识别也正常，但键盘背光无法按预设分区点亮。用户反馈左侧 WASD 区域持续熄灭，数字键区域亮度忽高忽低，部分机型在切换性能模式后会丢失自定义灯效。

## 现场现象

设备开机后，系统能识别内置键盘，所有按键扫描码均正常。按下背光快捷键时，屏幕会显示亮度调节提示，但键帽下方灯光没有同步变化。进入厂商控制中心后，软件界面可以保存灯效配置，但重启后配置恢复默认。外接键盘灯效正常，问题只出现在设备自带键盘背光组件。

## 排查过程

技术支持先排除键盘输入故障和接口识别异常，确认问题不是按键结构损坏，也不是 USB 供电或外设连接问题。随后检查 BIOS 中的键盘照明选项、嵌入式控制器固件版本、厂商控制中心服务状态以及键盘背光驱动日志。多台问题设备都出现背光控制器响应超时，但键盘输入控制器状态正常。

## 处理措施

对受影响设备先关闭厂商控制中心的自定义灯效，再恢复 BIOS 默认设置。若背光仍无法恢复，需要重新安装键盘背光控制组件，并执行嵌入式控制器断电复位。完成后分别测试常亮、呼吸、分区颜色和快捷键亮度调节，确认灯效配置在重启后仍可保留。

## 后续建议

客服记录工单时必须区分“键盘输入异常”和“键盘背光异常”。若按键输入正常但灯光分区、亮度、颜色或灯效配置异常，应收集 BIOS 版本、控制中心版本、背光驱动日志和故障照片，再转交二线硬件支持处理。
