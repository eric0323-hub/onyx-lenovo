### Symptom

When typing on the keyboard the customer may report random keys being displayed that are not being pressed.

This may be caused by incorrect routing of the keyboard ribbon cable.

![Cable](https://download.lenovo.com/km/media/images/HT514009/Keyboard)

### Applicable Brands

* Lenovo
* Ideapad

### Applicable Systems

* S145-14AST Laptop (ideapad) 81ST
* ideapad 3-14IGL05 Laptop 81WH
* S145-14API Laptop (ideapad) 81UV
* S145-14IIL Laptop (ideapad) 81W6
* V14-IIL Laptop (Lenovo) 82C4
* V14-ADA Laptop (Lenovo) 82C6
* S145-14IGM Laptop (ideapad) 81MW
* S145-14IKB Laptop (ideapad) 81VB

### Solution

Inspect the installation of the keyboard ribbon cable and correct if needed.

**Note:** If installed incorrectly, the adhesive backing will still be present on the keyboard ribbon cable.

1. Inspect the Keyboard ribbon cable after disassembling the system board and other components following the Hardware Maintenance Manual.

* If damage is present, replace the keyboard assembly (see picture below).
* If no damage is present, proceed to step 2.

![Cable](https://download.lenovo.com/km/media/images/HT514009/Cable)

2. Inspect the keyboard ribbon cable for an adhesive backing paper.

* If found, remove the adhesive backing paper (see picture below).
* If not found, continue with normal problem determination.

​![Step](https://download.lenovo.com/km/media/images/HT514009/Keyboard)

3. Fold the ribbon cable back and adhere it to the underside of the keyboard.

​![step](https://download.lenovo.com/km/media/images/HT514009/Keyboard)

4. After adhering the cable to the underside of the keyboard, apply Mylar sticker FRU# 5T10S33574 covering the touchpad screw that will be under the keyboard ribbon cable.

**Note**: A flat tool may be needed to remove the sticker from the backing.

![Sticker](https://download.lenovo.com/km/media/images/HT514009/Mylar) ![screw](https://download.lenovo.com/km/media/images/HT514009/Mylar)

5. Fold the ribbon cable back to avoid the stand-off on the underside of the keyboard and reattach to the system board.

![step](https://download.lenovo.com/km/media/images/HT514009/Keyboard)​

​![step](https://download.lenovo.com/km/media/images/HT514009/Keyboard)

6. Reassemble the system to complete the repair.

### Additional Information

This is a Fix on Contact solution that should be inspected / applied when performing repairs on an applicable system within its warranty period.

Include ‘HT514009’ in the failure description if submitting a warranty claim.
