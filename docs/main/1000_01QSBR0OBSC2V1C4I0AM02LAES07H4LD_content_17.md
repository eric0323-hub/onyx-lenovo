### Symptom

This document provides the following troubleshooting steps for the keyboard backlight.

For information about how to turn the keyboard backlight on or off, see [Is my PC configured with backlight keyboard and how to turn it on or off](https://support.lenovo.com/solutions/HT104656).

![Backlight](https://download.lenovo.com/km/media/images/HT117804/backlight two_20181019134040.JPG) ![Backlight](https://download.lenovo.com/km/media/images/HT117804/backlight 222_20181019134250.JPG)

Check to see if the keyboard backlight works after each step.

1. [Check whether the keyboard has a backlight](#step1)
2. [Configure Lenovo Vantage](#step2)
3. [Check whether keyboard backlight can be turned on under BIOS or Safe Mode](#step3)
4. [Perform a power drain](#step4)

### Applicable Brands

* ideapad
* ThinkPad

### Operating Systems

* Windows 10
* Windows 11

### Solution

**Step One - Check whether the keyboard has a backlight**

If there is a backlight icon on Esc or Space keys, then the keyboard is equipped with backlight hardware.

![Backlight button](https://download.lenovo.com/km/media/images/HT117804/backlight_20210310121530472.png)

![Backlight](https://download.lenovo.com/km/media/images/HT117804/e827a06ad70b4492af0618aa00143424.jpg)

![Escape button](https://download.lenovo.com/km/media/images/HT117804/backlight1_20210310123452888.png)

**Step Two - Configure Lenovo Vantage**

1. Search for [Lenovo Vantage](https://support.lenovo.com/solutions/ht505081).
2. Select the button on the top right corner. Select **Device**, **DEVICE SETTINGS**, **Input & Accessories**, and set the option under **Keyboard backlight**.  
   ![keyboard backlight in vantage](https://download.lenovo.com/km/media/images/HT117804/vantage_20250604073710854.png)

**Step Three - Check whether keyboard backlight can be turned on under BIOS or Safe Mode**

1. Enter ideapad [BIOS](https://support.lenovo.com/solutions/ht500216), ThinkPad [BIOS](https://support.lenovo.com/solutions/ht500222), or [Safe Mode](http://support.lenovo.com/solutions/HT116905).
2. Press Fn + Space or Esc to check if the backlight works.
   * If this works, go to [Lenovo support website](http://support.lenovo.com) to update BIOS to the latest.
   * If this does not work under BIOS or Safe Mode, back up important data and perform [System restore](https://support.lenovo.com/solutions/ht501548) from a previous restore point.

**Step Four - Perform a Power Drain**

For systems with a **detachable** battery, try the following steps:

* Unplug the AC adapter and remove the battery.
* Hold the Power button down for at least 10 seconds.
* Install the battery, plug in the AC adapter, and try powering on the system.

For systems with a **built-in** battery, try the following steps:

* Unplug the AC adapter.
* Hold the Power button down for at least 10 seconds.
* Reconnect the AC adapter and try turning it ON.

If the keyboard backlight can still not be turned on after trying the previous steps, click [Contact Us](http://support.lenovo.com/contactus).
