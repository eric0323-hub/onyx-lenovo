### Symptom

The 1,3,4,5,G,Z,C,V,B,Q,E,R,T,A,D,F keys may stop working, or when typing other keys, incorrect characters or actions may happen:

* "z" key outputs ZCVX
* "a" key tries to bookmark page
* "1" key outputs 1428709

### Applicable Brands

Chromebook

### Applicable Systems

100e Chromebook

### Operating Systems

Chrome OS

### Workaround

* [Reset Chromebook hardware:](https://support.google.com/chromebook/answer/3227606?hl=en)

1. Turn off Chromebook.
2. Press and hold Refresh ![Refresh](https://download.lenovo.com/km/media/images/HT507934/Chromebook Refresh_20190207210439312.png) + tap Power ![Power](https://download.lenovo.com/km/media/images/HT507934/Chromebook Power Button_20190207210506516.png).
3. When Chromebook starts up, release Refresh ![Refresh](https://download.lenovo.com/km/media/images/HT507934/Chromebook Refresh_2019020721054991.png).

* If the keyboard is not restored to normal operation, [Reset your Chromebook to factory settings](https://support.google.com/chromebook/answer/183084?hl=en) by performing a Powerwash.

If the above steps restore the keyboard to normal operation, report the problem to Google.

* Users: [Report a problem or send feedback about Chrome OS](https://support.google.com/chromebook/answer/2982029?hl=en&ref_topic=3417251)
* Administrators: [Contact Chrome Enterprise support](https://support.google.com/chrome/a/answer/4594885?hl=en&ref_topic=4386914)

If the above steps do not restore the keyboard to normal operation, contact Lenovo Service for additional support.

### Solution

Upgrade to the current version of the operating system to see if the issue is resolved. For more information, see [Update your Chromebook's operating system](https://support.google.com/chromebook/answer/177889?hl=en#:~:text=Check%20for%20updates%20yourself&text=At%20the%20bottom%20left%2C%20select,will%20start%20to%20download%20automatically.).
