## Symptom

Virtual keyboard function cannot be launched when logging in with a guest account.

## Applicable Systems

* Yoga Book 9 13IMU9 (Machine Type: 83FF)

## Workaround

Create a local account to serve as the guest account, since the virtual keyboard function is unavailable under the default guest account.

For information on how to create a local guest account, refer to the following **Create** video:  
<https://drive.google.com/file/d/10l-EuGDDDaXrn05x7j76OBIvJLLWG_tQ/view?usp=sharing>

The following **SignOut** video demonstrates that the virtual keyboard (OSK) works normally when logging in to the next account after logging out of the previous account:  
<https://drive.google.com/file/d/1SjFFpToWINiOcYKiC7eDzHqPfJJL5YMI/view?usp=sharing>
