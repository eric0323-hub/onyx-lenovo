### Symptom

Some keyboard keys (not function keys) do not perform the desired action when pressed. For function keys, make sure the hotkey software is up to date. See [Check Drivers](#Drivers) or [Function keys are not working](https://support.lenovo.com/solutions/HT510960). If specific keys seem to be swapped, see [Keyboard keys are incorrectly assigned - Windows 10, 11 - ThinkPad](https://support.lenovo.com/solutions/ht104229).

To fix this issue, try the following troubleshooting methods. If the issue is resolved, no need to proceed to the next method.

### Applicable Brands

* ideapad
* ThinkPad

### Operating Systems

* Windows 10
* Windows 11

### Solution

If none of the steps resolve the issue, see [Contact us](http://support.lenovo.com/contactus) for further assistance.

#### Step One - Test if On-Screen Keyboard is workable

1. While on the Log in Screen, click the Ease of access logo.  
     
   ![Ease of access](https://download.lenovo.com/km/media/images/HT103985/log in page_20170215072048.JPG)
2. Place a check in the **Type without the keyboard (On-Screen Keyboard)** box.  
     
   ![On-Screen Keyboard](https://download.lenovo.com/km/media/images/HT103985/OSK_20170215072243.PNG)
3. By using your mouse cursor, click and check that all keys are working fine with the On-screen keyboard. Also check the keys that are not working with the physical keyboard.  
     
   ![Check keys](https://download.lenovo.com/km/media/images/HT103985/On-ScreenKeyboardPortable2_20170215072936.JPG)  
   **Note:** An alternate way to get to the on-screen keyboard if logged in is to select the Windows icon, **Settings**, **Ease of Access**, **Keyboard**. Set the toggle under **Use the On-Screen Keyboard** to **On**.


#### Step Two - Disable all Keyboard shortcuts and enhancements

If the computer does not have a password (or use the on-screen keyboard to enter a password, see [Use the On-Screen Keyboard (OSK) to type](https://support.microsoft.com/windows/use-the-on-screen-keyboard-osk-to-type-ecbb5e08-5b4e-d8c8-f794-81dbf896267a)) and can still boot inside Windows, proceed with the following settings:

1. Search for and select **Control Panel**.  
   ![Control Panel](https://download.lenovo.com/km/media/images/HT103985/controlpanelWin11_20250501173847367.png)
2. Select **Ease of Access**\**Ease of Access Center**\**Make the keyboard easier to use** (Windows 11: **Ease of Access Center**/**Make the keyboard easier to use**).  
     
   ![Control panel](https://download.lenovo.com/km/media/images/HT103985/make the keyboard easier to use_20170215074900.JPG)
3. Uncheck all the boxes and apply the settings, and restart the computer.  
     
   ![Enhancements](https://download.lenovo.com/km/media/images/HT103985/keyboard enhancements_20170215075111.JPG)  
   
   Windows 11  
   ![Windows 11 keyboard settings](https://download.lenovo.com/km/media/images/HT103985/Windows11makekeyboardeasiertouse_20250312130404114.png)


#### Step Three - Test keys with Notepad

1. Search for and select **Notepad.  
   ![Search icon](https://download.lenovo.com/km/media/images/HT103985/Searchicon2_2022020917582774.png)**
2. Try typing the keys and see if they work or are displayed in Notepad. If the failed keys work, this may be a software issue. Test the failed keys with different software installed on the PC. If the keys fail with a specific software application, refer to the vendor's software instructions and reset the settings.


#### Step Four - Check Drivers

Reinstall the keyboard driver.

1. Search for and select **Device Manager**.  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT103985/Windows11Devicemansearch_20250909144414130.png)
2. Expand the keyboards section and find the driver.
3. Right-click the driver.
4. Select **Uninstall device**.  
   ![Device Manager](https://download.lenovo.com/km/media/images/HT103985/devicem1_20210111185135735.png)
5. Reboot (power off and power on the machine), and let Windows install the driver.
6. If this does not work, try updating drivers.

Make sure drivers are up to date.

1. Go to the web site: <https://support.lenovo.com>.
2. Select **Detect Product**.  
   ![Detect Product](https://download.lenovo.com/km/media/images/HT103985/detectproduct2024_20240703141130427.png)
3. Select **Drivers & Software**.
4. Select **Scan Now** under **Automatic Update** or check for keyboard updates under **Manual Update**.  
   ![Automatic Update](https://download.lenovo.com/km/media/images/HT103985/automaticdriverupdateJan2023_20230206162134211.png)

How to roll back a driver if the issue occurred after a recent update.

1. Find the driver in [Device Manager](https://support.microsoft.com/windows/open-device-manager-a7f2db46-faaf-24f0-8b7b-9e4a6032fc8c).
2. Right-click the driver and select **Properties**.
3. Select the **Driver** tab and select **Roll Back Driver** if the option is available. If Roll Back Driver is grayed out, this means no earlier versions were installed.
4. Reboot the PC.


#### Step Five - Disable gaming mode

If gaming mode is enabled, try disabling gaming mode. For Windows 11, see [How to access the Gaming section in Windows 11](http://support.lenovo.com/solutions/HT517059).

1. Open the Windows Settings (**Start** and **Settings**).
2. Select the **Gaming** section.
3. On the left hand menu, select **Game Mode** and toggle the option to **Off**.  
   ![Game Mode Toggle](https://download.lenovo.com/km/media/images/HT103985/Game Mode toggle_2022121620450393.png)


#### Step Six - Try Keyboard Troubleshooting

Use keyboard troubleshooting. For more details, see [How to run Windows Troubleshooters - Windows 10 and Windows 11](https://support.lenovo.com/solutions/HT515094).

Windows 10

1. Right-click the Windows icon.
2. Select **Settings**.  
   ![Settings](https://download.lenovo.com/km/media/images/HT103985/Settings4_20220608153212871.png)
3. Select **Update & Security**.  
   ![Update and Security](https://download.lenovo.com/km/media/images/HT103985/updateandsecurity_2024100213240848.png)
4. Select **Troubleshoot**. Select **Keyboard** to fix problems with keyboard settings.  
   ![Keyboard](https://download.lenovo.com/km/media/images/HT103985/keyboard_20200408162452716.png)

Windows 11

1. Select **Start** and **Settings**. **Note**: Newer versions of Windows 11 may no longer have certain troubleshooters. For more information, see [Deprecation of Microsoft Support Diagnostic Tool (MSDT) and MSDT Troubleshooters](https://support.microsoft.com/en-us/windows/deprecation-of-microsoft-support-diagnostic-tool-msdt-and-msdt-troubleshooters-0c5ac9a2-1600-4539-b9d0-069e71f9040a).  
   ![Settings](https://download.lenovo.com/km/media/images/HT103985/Windows11startsettings_2025071019091877.png)
2. Select **System** and **Troubleshoot**.
3. Select **Other troubleshooters**.
4. Select **Run** next to **Keyboard**.  
   ![Keyboard](https://download.lenovo.com/km/media/images/HT103985/troubleshooterWin11B_20230508152659423.png)


#### Step Seven - Check if the key works under Command Prompt

1. Enter Command Prompt.
   * Press Windows + X, then choose **Windows PowerShell (Admin)** or **Command Prompt** (depending on the version of Windows 10) or search for command prompt, and then select **Command Prompt**.  
     ![Command Prompt](https://download.lenovo.com/km/media/images/HT103985/ps_20200409134052945.png)
2. Under the **Administrator: Command Prompt** window,test the failed keys.
   * If the failed keys work, the issue should be caused by software, not PC hardware. Test the failed keys with different software installed on the PC. If the keys fail with a specific software application, refer to the vendor's software instructions and reset the settings.
   * If the failed keys do not work, the issue may be with the PC hardware. Please [Contact us](http://support.lenovo.com/contactus) on the support website. Lenovo local service technicians will provide further diagnosis.


#### Step Eight - Clean the Computer

Check if any foreign objects have fallen on the keyboard or if any objects are lodged in the keyboard. Shut down the PC and gently turn over to dislodge debris. See this link for more information, [Keeping your computer clean – Notebooks, All-In-One Desktops, Tiny-In-One, and Monitors](https://support.lenovo.com/solutions/HT035676).



#### Step Nine - Check Typing Settings

1. Select **Start** and **Settings**.  
   ![Settings](https://download.lenovo.com/km/media/images/HT103985/Settings4_20230109141148934.png)
2. Select **Devices**.
3. Select **Typing**.
4. Set settings back to the default options if they were recently changed.  
   ![Typing](https://download.lenovo.com/km/media/images/HT103985/typingkeyboard_20230109140242438.png)


#### Step Ten - Perform System Restore

Consider a system restore if your keyboard recently stopped working all of a sudden.

System Restore is a feature that allows you to revert your computer’s state to a previous point in time. If an install failure or data corruption occurs, System Restore can help you return a system to working condition without having to reinstall the operating system.

Restore Windows System by Creating Restore Points：

For information about restoring or reinstalling your operating system, refer to [How to restore the system to a system restore point](https://support.lenovo.com/solutions/ht501548).



#### Step Eleven - General External Keyboard Troubleshooting Steps

Try the following if an external keyboard is not working.

* Check the wireless connection for a wireless keyboard.
* If the keyboard requires pairing, try the pairing steps again.
* If the keyboard requires batteries, check the batteries.
* Try a different keyboard.
