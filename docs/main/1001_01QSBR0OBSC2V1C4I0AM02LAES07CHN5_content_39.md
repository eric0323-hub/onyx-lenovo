## Symptom

Lenovo User center apps stop working on Yoga Book 9, following a system update.

## Applicable Systems

Yoga Book 9 (83FF)

## Solution

1. Go to folder **C:\Program Files\Lenovo\YB9App**
2. Run **unins000.exe** to uninstall the program.
3. Download the **new version** from this link: <https://einkgen2-download.lenovo.com/YB9Update/YogaBook_Solutions_Setup_3.1.0.22_TrialVersion.exe>
   
   Run the installer and allow **UAC** to proceed with installing.
