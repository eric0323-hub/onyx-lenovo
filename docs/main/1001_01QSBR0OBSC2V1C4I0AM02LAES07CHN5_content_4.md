### Symptom

The function keys are not working. For standard keyboard keys, see [Keyboard keys may not work](https://support.lenovo.com/solutions/ht103985).

### Applicable Brands

* ThinkPad
* ideapad

### Operating Systems

* Windows 10
* Windows 11

### Solution

Try the following options:

#### Check the User Guide

Check the User Guide and see what the function keys should do. Make sure nothing is preventing the function keys from working. For example, if the function key for increasing the volume does nothing, make sure the speakers are not muted.

Search for function keys in the manual, [How to find and view manuals for Lenovo products - ThinkPad, ThinkCentre, ideapad, ideacentre](https://support.lenovo.com/solutions/HT077589). The following example is for a ThinkPad T480s.

![Special keys](https://download.lenovo.com/km/media/images/HT510960/special_20200811155813108.png)

![Function keys](https://download.lenovo.com/km/media/images/HT510960/special1_20200811155838174.png)

![Function keys](https://download.lenovo.com/km/media/images/HT510960/special2_20200811155915225.png)



#### Make sure hotkey software is up to date

Updates are available with the automatic scan option:

1. Go to [https://support.lenovo.com](https://support.lenovo.com/).
2. Select **Detect Product**.  
   ![Detect Product](https://download.lenovo.com/km/media/images/HT510960/detectproduct2024_20241121135008650.png)
3. Select **Drivers & Software**.  
   ![Drivers and Software](https://download.lenovo.com/km/media/images/HT510960/driversandsoftware2024_2024112113510073.png)
4. Select **Scan Now** under **Automatic Update**.  
   ![Automatic Update](https://download.lenovo.com/km/media/images/HT510960/automaticdriverupdateJan2023_20240222150537496.png)


#### Check settings in Control Panel

Check settings in Control Panel (this information is in the User Guide).

1. Search for and select **Control Panel**.  
   ![Control Panel](https://download.lenovo.com/km/media/images/HT510960/control_202211041413332.png)  
   ![Control Panel - Windows 11](https://download.lenovo.com/km/media/images/HT510960/controlpanelWin11_20250121163335358.png)
2. Change the view from Category to **Large icons** or **Small icons**.
3. Select **Lenovo - Keyboard Manager** and the Keyboard Manager window should be displayed.
4. Make sure the settings are correct (for example, no special mappings that might prevent the expected behavior).  
   ![Keyboard Manager](https://download.lenovo.com/km/media/images/HT510960/keyboardm_20200811162455101.png)


#### Check settings in Lenovo Vantage

If Lenovo Vantage is installed on the system, check the User defined key settings and make sure the settings are as expected. These settings are located under **Device settings** > **Input** > **User defined key**.

![Vantage](https://download.lenovo.com/km/media/images/HT510960/Vantage_20250806064809241.jpg)



#### Make sure the function keys are not locked

The User Guide should have information about what keys can be used to lock the function keys. If the lock is off, then the default function should work.



#### Check any function key settings in the BIOS

The BIOS options should be similar to the Lenovo - Keyboard Manager options.

* [Recommended way to enter BIOS for Think Series](https://support.lenovo.com/solutions/ht500222)
* [Recommended way to enter BIOS for IdeaPad, Lenovo Laptops](https://support.lenovo.com/solutions/ht500216)
* [Recommended way to enter BIOS for Lenovo Desktops and All-in-Ones](https://support.lenovo.com/solutions/ht500217)
* [BIOS Simulator](https://download.lenovo.com/bsco/index.html)

Keyboard settings can often be found in the BIOS under **Config** and **Keyboard/Mouse**.

* Check Fn and Ctrl Key swap (Disabled)
* Check Fn Sticky Key (Disabled)
* Check F1-F12 as Primary Function (Enabled)  
  ![Keyboard, Mouse](https://download.lenovo.com/km/media/images/HT510960/BIOSt480skeyboard_20230906144235985.png)
