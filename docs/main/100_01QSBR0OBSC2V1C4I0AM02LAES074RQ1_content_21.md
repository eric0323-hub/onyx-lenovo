### Symptom

Applications displayed in an external monitor are not restored when the external monitor is disconnected during sleep or hibernation.

The following steps reproduce the issue:

1. Connect an external monitor to a system.
2. Open an application or file, such as Microsoft Edge, Lenovo Vantage, or Control Panel.
3. Change the project mode to Extend.
4. Drag the application or file from the system LCD to the external monitor.
5. Put the system into sleep or hibernation.
6. Disconnect the monitor. Bring the system out of sleep or hibernate (move the mouse or press a key).
7. The application or file is minimized.
8. Click the minimized application or file on the taskbar.
9. The application or file cannot be restored.
10. Connect the external monitor again.
11. The application or file works.

### Applicable Brands

ThinkPad

### System Is Configured With

External monitor

### Operating Systems

Windows 11

### Solution

Use one of the following solutions to resolve the issue:

* Connect the display again.
* Close applications displayed in an external monitor before the system goes into sleep or hibernation.
* Move applications displayed in the external monitor to the integrated monitor before putting the system into sleep or hibernation.

### Additional Information

This is a known issue in Windows 11 and Microsoft is working on a resolution.
