### Symptom

One or more of the following conditions may occur on the T470p system:

* POST message at boot stating the AC adapter is not the correct wattage for the system.
* AC adapter not recognized in the operating system.
* Main battery will not charge.

### Applicable Brands

ThinkPad

### Applicable Systems

ThinkPad T470p (20J6, 20J7)

### Operating Systems

Windows 10 64-bit

### Solution

A BIOS update was released to fix the reported conditions, see [BIOS Update Utility for Windows 10 (64-bit) - ThinkPad T470p](https://support.lenovo.com/downloads/DS120707).

Be sure the system is at version 1.22 (R0FET42W) - 1.05 (R0FHT17W) or later ([How to check your BIOS version on Windows 10](https://support.lenovo.com/solutions/HT070218)).

For BIOS information, click one of the following:

* [Video] [How-To – Enter The BIOS Setup Utility in Windows Windows 10](https://www.youtube.com/watch?v=iKKvOS6NE1k)
* [How to update system BIOS](https://support.lenovo.com/solutions/HT500008)
* [Lenovo System Update: Update Drivers, BIOS and Applications](https://support.lenovo.com/solutions/ht003029)

### Additional Information

The BIOS update should be tried first before replacing any parts. The battery and AC adapter charger should not need replacing.

For any system where the BIOS/EC update does not resolve the reported conditions:

1. Replace the systemboard (under warranty terms).
2. Be sure BIOS/EC version is at 1.22 (R0FET42W) - 1.05 (R0FHT17W) or later.
