# **Wired connection drops**

![](https://ukm.lenovo.com/prod-api/file/download?key=2025/04/29/Lenovo_Logo_20250429174901A001.png)

**Author:** System

1. Navigate to Device Manager>Network adapters and get properties on the network adapter
2. Go to the advanced tab
3. Set “Energy Efficient Ethernet” to “Off”
4. \* For RealTek this is “Green Ethernet”
5. And “System Idle Power Saver” to “Disabled”
6. \* May also be labeled as “Idle Power Saving” or “Power Saving Mode”
